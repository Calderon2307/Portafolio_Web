# Proyecto 9: Questions API (JS)

## Descripción del Proyecto

El proyecto "Questions API JS" es una aplicación frontend desarrollada para consumir la API de preguntas y respuestas "OTDB" (Open Trivia Database). Esta aplicación ofrece a los usuarios una experiencia interactiva de trivia, presentando 10 preguntas aleatorias con diferentes niveles de dificultad y temas. Los usuarios pueden responder a cada pregunta y recibir retroalimentación inmediata sobre la corrección de su respuesta. En caso de respuestas incorrectas, se muestra la respuesta correcta.

La aplicación lleva un conteo en tiempo real de las preguntas respondidas correctamente, brindando a los usuarios una visión clara de su desempeño durante la sesión de trivia. Además, utiliza JavaScript para manejar la lógica de interacción con la API, la presentación de las preguntas y respuestas, así como la actualización del marcador de puntos.
